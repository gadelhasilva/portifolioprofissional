☁️ Infraestrutura como Código & CI/CD (Este Portfólio)

Este site não é apenas uma página estática, é uma prova de conceito de práticas DevOps. Ele está hospedado em uma instância EC2 (Amazon Linux 2023) na AWS com Nginx. A entrega contínua (Deploy) é feita de forma 100% automatizada via GitHub Actions, utilizando um usuário de serviço (`deployer`) configurado com chaves SSH baseadas no princípio do menor privilégio.
